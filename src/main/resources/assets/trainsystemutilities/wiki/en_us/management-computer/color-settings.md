# Color Settings

```embed:color-settings
```

The Color Settings popup of the Management Computer. Customizes the color of each element on the monitor display.

[[TOC]]

## How to open

1. **Place** a **Management Computer** block and **right-click** it to open the screen.
2. **Click the "🎨 Color" button at the bottom** of the screen to open the monitor color settings popup.

## Elements you can recolor (targets)

**Click the "Target:" dropdown** at the top of the popup and **click** the element you want to recolor. Available elements:

| Target | Description |
|---|---|
| Title | Panel title text color |
| Border | Panel border color |
| Train name | Train-name text color |
| Speed / Status | Speed and running-status text color |
| Destination | Destination text color |
| Clock | Clock display color |
| Stat value | Color of numbers such as station and train counts |
| Signal (green) | Color of green signals on the map |
| Signal (red) | Color of red signals on the map |
| Map line | Color of the lines on the map |
| Map station | Color of the station icons on the map |
| Map train | Color of the train icons on the map |

## Usage

1. **Pick a target**: click the "Target:" dropdown → click an element from the list above.
2. **Pick a color**: **click one of the preset colors** below directly, and it is applied to the selected target. "Current:" shows the present color.
3. **Revert**: **click the "Reset" button** to return the selected target to its default color. **Click the "Reset all" button** to return every target to its default color.
4. **Close**: **click the ✕ (close) button** at the top right of the popup.

## Presets

12 preset colors can be applied in one touch by **clicking**.

## Scope

Color settings are per Management Computer.  
The same colors are applied to every linked monitor.
