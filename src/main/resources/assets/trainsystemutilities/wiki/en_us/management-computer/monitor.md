# Monitor Display

![](bws:trainsystemutilities:wiki/screens/management-computer__map__ja_jp.png)

The Monitor Display tab of the Management Computer. Shows a custom layout on linked nearby monitor blocks.

[[TOC]]

## How to open

1. **Place** the **Management Computer** block and **right-click** it to open the screen.
2. The monitor features are operated from the **monitor toggle at the top** of this screen and the **buttons at the bottom** (🎨 Color / ▒ Layout) (see [Controls](#操作) below).

## Features

- Custom layout editing ([Layout Editor](layout-editor.md))
- Color customization ([Color Settings](color-settings.md) / [Color Picker](symbol-editor.md#color-picker))
- Monitor linking ([Monitor Link Card](../tools/monitor-link-card.md))
- Batch apply / face-flip / private/public

## Controls {#操作}

| Element | Action | Function |
|---|---|---|
| Monitor toggle | **Click** to turn ON/OFF (ON = green) | When ON, starts displaying on linked monitors ([workflow](#モニター連携の流れ)) |
| 🎨 Color button (bottom) | **Click** | Opens the [Color Settings](color-settings.md) popup |
| ▒ Layout button (bottom) | **Click** | Opens the [Layout Editor](layout-editor.md) |
| Create line symbols | Select the **Ⓜ Line Symbols** tab in the top-left dropdown | Create and edit symbols in the [Line Symbol Editor](symbol-editor.md) |

> [!NOTE]
> Even with the monitor toggle ON, there is nowhere to display unless you have first linked a monitor block with the [Monitor Link Card](../tools/monitor-link-card.md). Complete the [monitor linking workflow](#モニター連携の流れ) below first.

## Monitor linking workflow {#モニター連携の流れ}

1. **Place** the Management Computer.
2. **Place** monitor blocks nearby.
3. Hold the [Monitor Link Card](../tools/monitor-link-card.md), **right-click** a monitor block → **right-click** the Management Computer to link.
4. Open the Management Computer and **click the monitor toggle to turn it ON** (green) → instant sync to the linked monitors.
