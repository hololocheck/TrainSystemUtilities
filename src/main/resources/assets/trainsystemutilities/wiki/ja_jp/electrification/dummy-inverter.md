# ダミーインバーター

```embed:item id=trainsystemutilities:fe_inverter_dummy size=48 label=true
```

FE 集電 / 駆動システムを **使わずに、パンタグラフを純粋な装飾** として使いたいプレイヤー向けの "見た目だけ" の FE インバーター変種。

[[TOC]]

## 概要

見た目・配置挙動は本物の [FE インバーター](fe-inverter.md) と完全に同一の 3 連マルチブロックですが、**内部に FE バッファを持たず**、列車に搭載しても電気系の処理は一切行いません。

## できること / できないこと

| 項目 | 本物 FE インバーター | ダミーインバーター |
|---|---|---|
| 見た目 (モデル / テクスチャ) | ✅ | ✅ (同一) |
| 3 連マルチブロック配置 | ✅ | ✅ (同一) |
| 管理 UI の電化情報パネル表示 | ✅ | ✅ |
| UI からのパンタ展開 / 折畳 | ✅ | ✅ |
| 架線への接触判定 (バー押し下げ視覚) | ✅ | ✅ |
| FE 集電 (架線から電力受け取り) | ✅ | ❌ |
| FE 維持コスト消費 | ✅ | ❌ |
| パンタ全折畳での強制停止 | ✅ | ❌ |
| 隣接 FE 機械への給電 | ✅ | ❌ |

## 用途

### 1. 列車屋根のパンタ装飾

`[車両屋根: パンタグラフ + 装飾インバーター]`

→ 列車を電化規格に見せたいが、ゲームプレイ上 Create エネルギー駆動のままにしたい。
管理 UI からパンタ展開 / 折畳が可能なので、駅停車中だけパンタを下ろすといった演出ができます。

### 2. 線路脇の装飾オブジェクト

車両に乗せず単独で配置すれば、ホームの機器箱・変電施設のダミーパーツ・架線下の地上機器など、純粋な装飾ブロックとして使えます。右クリックしても何も起きません。

## 設置方法

1. クリエイティブインベントリ TSU タブ、または `/give @s trainsystemutilities:fe_inverter_dummy` で入手。
2. 手に持って置きたい場所を **右クリック** で設置。本物の FE インバーターと同じ 3 連機器です。
3. 設置時はクリックした位置が **中央 (CENTER)** になり、向いている方向の前後 (HEAD / TAIL) に自動で 3 ブロック分配置されます。
4. 前後どちらかに空きがないと設置できません。
5. **素手で右クリックしても何も起きません** (装飾ブロックのため。状態表示もありません)。

## 列車に搭載した時の挙動

```
[車両 1: パンタグラフ + ダミーインバーター]
       ↓
   管理 UI に電化情報パネル表示 (FE 表示は "装飾モード")
       ↓
   UI から「全パンタ展開」「全パンタ折畳」操作可能
       ↓
   架線下を通過 → パンタのバーが架線に押し下げられる視覚効果
       ↓
   FE 集電は発生しない (装飾モード)
```

> [!NOTE]
> **本物の FE インバーターと混在もできます。**
> 同じ列車内に本物 1 台 + ダミー複数台を載せた場合、列車全体としては「電化列車」として扱われ、本物側だけが集電・FE 管理を行います。ダミーはあくまで装飾枠を増やすだけです。

## 注意

- ダミーインバーターを **唯一のインバーターとした列車** は、架線下を走行しても FE は一切貯まりません。電力で走らせたい場合は本物の [FE インバーター](fe-inverter.md) を 1 台以上搭載してください。
- 管理 UI 上で本物との区別は「電化詳細」ダイアログの該当車両行に **`(装飾モード)`** バッジが表示されるかどうかで判断できます。
