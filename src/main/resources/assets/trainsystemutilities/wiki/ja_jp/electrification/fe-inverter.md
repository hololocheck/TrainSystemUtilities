# FE インバーター

```embed:item id=trainsystemutilities:fe_inverter size=48 label=true
```

列車車両に乗せる FE バッファブロック。パンタグラフからの集電を蓄電 + 他 mod の FE 機械に給電する。

[[TOC]]

## 設置方法

FE インバーターは **3 ブロック連結の機器** です。手に持って設置したい場所を **右クリック** すると、クリックした位置が中央になり、その前後に自動で 3 ブロック分が配置されます。

1. インバーターを手に持ち、車両の床下などに置きたい場所を **右クリック**。
2. **向いている方向の前後に 3 ブロック分の空きが必要** です。空きがないと設置できません。
3. 3 連のどこか 1 ブロックを壊すと、3 ブロックまとめて撤去されます。
4. パンタグラフを別車両に乗せても、同じ列車の FE インバーターと自動で電気共有します。
5. インバーターから IEnergyStorage capability で隣接ブロックに FE 出力します。

> [!TIP]
> 設置した FE インバーターを **素手で右クリック** すると、現在の FE 残量 / 容量と「駆動可 / 駆動不可」がチャットに表示されます (状態確認用)。

## バッファ仕様

| 項目 | 値 |
|---|---|
| 容量 | 1,000,000 FE |
| 入力レート | 10,000 FE/tick (パンタグラフ + 隣接 FE 入力合算) |
| 出力レート | 10,000 FE/tick (隣接 FE 出力) |

## 列車プール電化

複数連結時の挙動 (= [パンタグラフ](pantograph.md#複数連結時の挙動) と関連):

```
[車両 1: パンタグラフ]   [車両 2: FE インバーター]   [車両 3: FE インバーター]
       ↓                          ↓                          ↓
       └─────────────── 列車プールで FE 共有 ───────────────┘
```

- パンタグラフが車両 1 にあれば、車両 2 / 車両 3 の FE インバーターも給電対象
- どの FE インバーターも隣接の FE 機械に出力可能
- バッファ合算: 3 個ある場合 = 300 万 FE 容量

## 連携できる外部 mod

| mod | 接続方法 |
|---|---|
| **Mekanism** | Universal Cable / Ultimate Energy Cube などで隣接接続 |
| **Applied Energistics 2** | Energy Cell / Energy Acceptor で接続 |
| **Industrial Foregoing** | Power Conduit で接続 |
| **Create** (Electric Engine 経由) | 隣接で FE → Rotational Force 変換 |
| その他 FE 互換 mod | IEnergyStorage 経由で全て接続可能 |

## GUI / 状態確認

FE インバーター本体には専用 GUI はない (シンプルなバッファブロック)。状態は次の 2 通りで確認できます。

- **設置したインバーターを素手で右クリック** → その 1 台の FE 残量 / 容量をチャット表示。
- **管理用コンピューター > 列車詳細 > 電化詳細** → 列車全体の蓄電量をまとめて確認 ([電化詳細 popup](../management-computer/trains.md))。
- **[電力チェッカー](power-checker.md) で右クリック** → 設置時は単体情報、列車組込時は所属列車の合計蓄電を表示。

> [!TIP]
> 列車が長距離区間でチャンク外に出ても、FE インバーターのバッファ残量はサーバーで保持される (= チャンクロード非依存)。

## モデル

vanilla block model (= JSON 定義)。テクスチャは黒 + 緑の LED 風アクセント。  
Geckolib ではないので軽量 (シンプルな立方体形状)。
