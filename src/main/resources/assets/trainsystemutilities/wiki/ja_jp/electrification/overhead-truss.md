# 架線トラス (Overhead Truss)

```embed:item id=trainsystemutilities:overhead_truss size=48 label=true
```

複数の線路をまたぐ門型の架線支持構造物です。複線以上の区間で、1 本の構造物から複数の架線を吊り下げられます。

[[TOC]]

## 設置

架線トラスはインベントリから手に持ち、**置きたい場所を右クリック** で設置します。

- 向きは 8 方向 (45° 刻み) に対応し、斜め区間にも沿わせられます。
- **横に伸ばしたいとき (梁を渡す)**: すでに置いてあるトラスを右クリックすると、その梁の向きの先端にもう 1 個トラスが継ぎ足されます。連続で右クリックすれば線路群をまたぐ長さまで伸ばせます。
- **斜めの架線柱に付けるとき**: 斜めに立てた [架線柱](overhead-pole.md) の側面を右クリックすると、柱の向きに合わせた斜め方向のマスにトラスが配置されます。
- 大量設置は [架線柱自動配置ツール](../tools/overhead-pole-auto-tool.md) が便利です (設定画面で「トラスを配置」を ON にすると、柱と一緒にトラスも自動で並びます)。

> [!IMPORTANT]
> トラス自体には架線を直接張れません。トラスの側面や下面に [架線碍子](insulator.md) を右クリックで設置し、その碍子を [架線接続ツール](wire-connector.md) で右クリックして架線を張ります。

## 架線を張るまでの流れ

1. 線路群をまたぐようにトラスを設置 (梁を右クリックで継ぎ足して必要な幅にする)。
2. 各線路の真上あたりに [架線碍子](insulator.md) を右クリックで設置。
3. [架線接続ツール](wire-connector.md) を設置モードにして、**碍子 → 碍子** の順に右クリックで架線を張る。

> [!NOTE]
> 単線なら [架線柱](overhead-pole.md) の方が省スペースです。
