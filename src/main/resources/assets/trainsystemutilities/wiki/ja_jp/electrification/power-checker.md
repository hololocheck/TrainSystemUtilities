# 電力チェッカー

```embed:item id=trainsystemutilities:power_checker size=48 label=true
```

電化システムの **デバッグ・状態確認用ツール**。対象ブロックを右クリックすると、その電化状態をチャットに表示する。

[[TOC]]

## 使い方

電力チェッカーを **手に持って**、調べたい装置を **右クリック** するだけです。GUI は開かず、結果はチャットに文字で表示されます。対象になるのは電化システムのブロックだけで、それ以外を右クリックすると「対象外」と表示されます。

対応する対象と表示内容:

| 対象 | 表示される情報 |
|---|---|
| **架線碍子** | この碍子に attach している全架線数 / 通電中の本数 / 各架線の通電 ON / OFF |
| **変電所キュービクル** (本体 / ダミーどちらでも) | バッファ FE 残量・容量・通電ネットワーク (碍子) 数 |
| **FE インバーター** (設置時) | バッファ FE 残量・容量 |
| **FE インバーター** (列車組込時) | 所属列車の合計 storedEnergy・全インバータ合計容量 |
| **パンタグラフ** (設置時 / 列車組込時) | 接触中の架線セグメント・現 tick の集電量 |
| その他のブロック | 「対象外」と表示してパス |

> [!TIP]
> 表示は **チャット (アクションバー not)** に送られる。サーバー / シングルプレイ問わず動作。
> 列車組込時の数値は server tick ベースなので、走行中の値も正確。

## 出力例

```
[Insulator @ (123, 65, -42)]
  接続架線数: 3
  通電中: 2 / 3
  ▸ wire 1: ON (to insulator @ (118, 65, -42))
  ▸ wire 2: ON (to insulator @ (128, 65, -42))
  ▸ wire 3: OFF (to insulator @ (123, 65, -50))  ← 切れた電源系統
```

```

  通電碍子: 6 個
  通電架線: 14 本
```

```
[FE Inverter (Train #train_jb_03)]
  列車 storedEnergy: 12,400 / 80,000 FE (15.5%)
  搭載インバータ: 4 個 (各 20,000 FE 容量)
```

## トラブルシューティング用途

| 症状 | チェッカーで確認 |
|---|---|
| 「列車が動かない」 | パンタを叩いて集電量を確認 → 0 なら架線が通電してない |
| 「架線が暗い」 | 端の碍子を叩いて通電ネットワーク確認 → ハブまで辿る |
| 「変電所に FE が入ってこない」 | 変電所キュービクルを叩いてバッファ残量と入力レート確認 |
| 「列車の蓄電がすぐ無くなる」 | インバータを叩いて消費 vs 集電のバランス確認 |
