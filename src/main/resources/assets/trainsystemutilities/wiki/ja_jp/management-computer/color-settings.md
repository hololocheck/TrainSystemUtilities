# 色設定

```embed:color-settings
```

管理用コンピューターの色設定 popup。モニター表示の各要素ごとに色をカスタムする。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを **設置** して **右クリック** で画面を開く。
2. 画面 **下部の「🎨 色」ボタンをクリック** すると、モニター色設定 popup が開きます。

## 色を変える要素（対象）

popup 上部の **「対象:」ドロップダウンをクリック** し、色を変えたい要素を **クリックで選びます**。選べる要素:

| 対象 | 説明 |
|---|---|
| タイトル | パネルのタイトル文字色 |
| 枠線 | パネルの枠線色 |
| 列車名 | 列車名の文字色 |
| 速度/状態 | 速度・運行状態の文字色 |
| 目的地 | 行き先の文字色 |
| 時計 | 時刻表示の色 |
| 統計値 | 駅数・列車数などの数値色 |
| 信号(緑) | マップの青信号の色 |
| 信号(赤) | マップの赤信号の色 |
| マップ線 | マップの路線の色 |
| マップ駅 | マップの駅アイコンの色 |
| マップ列車 | マップの列車アイコンの色 |

## 操作

1. **対象を選ぶ**: 「対象:」ドロップダウンをクリック → 上の一覧から要素をクリック。
2. **色を選ぶ**: 下に並ぶ **プリセット色を直接クリック** すると、選んだ対象にその色が適用されます。「現在:」に今の色が表示されます。
3. **元に戻す**: **「リセット」ボタンをクリック** で選択中の対象を既定色に戻す。**「全リセット」ボタンをクリック** で全対象を既定色に戻す。
4. **閉じる**: popup 右上の **✕（閉じる）ボタンをクリック**。

## プリセット

12 色のプリセットを **クリック** でワンタッチ適用できます。

## 適用範囲

色設定は管理用コンピューター単位。  
リンクされた全モニターに同色が適用される。
