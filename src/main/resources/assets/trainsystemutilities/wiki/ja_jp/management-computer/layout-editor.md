# レイアウトエディタ

```embed:layout-editor
```

管理用コンピューターのレイアウトエディタ。モニター表示に、どのパネル（路線マップ・列車一覧・時計など）をどこにどの大きさで並べるかを編集する。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを **設置** して **右クリック** で画面を開く。
2. 画面 **下部の「▒ レイアウト」ボタンをクリック** すると、レイアウトエディタが開きます。
3. エディタ中央には、実際のモニターに見立てた **プレビュー枠** があり、左側（または上部）に置けるパネルの **パレット** が並びます。

## 置けるパネル（パレット）

パレットには次のタイルが並びます。これらを **プレビュー枠にドラッグ** して配置します。

| タイル | 内容 |
|---|---|
| 🗺 路線マップ | 路線網のマップ |
| 🚂 列車 | 列車一覧 |
| 🕒 時刻表 | スケジュール |
| 🏯 駅数 | 駅の総数 |
| 🚆 列車数 | 列車の総数 |
| 🚦 信号数 | 信号の総数 |
| 🕓 時計 | 時刻表示 |

## 操作

プレビュー枠の中で、パネルを置く・動かす・大きさを変える、を次のように行います。

| 操作 | やり方 | 動作 |
|---|---|---|
| パネルを追加 | パレットのタイルを **押したままプレビュー枠へドラッグして離す** | その位置にパネルが追加される |
| パネルを移動 | プレビュー内のパネルを **左ドラッグ** | 位置を動かす（枠内に収まるよう自動調整） |
| パネルの大きさ変更 | パネルに **マウスを乗せてマウスホイール**（上=拡大 / 下=縮小） | パネルを中心基準で拡大縮小 |
| パネルを選択 | パネルを **クリック** | 選択状態にする |
| パネルを削除 | 選択して **Delete キー** を押す | 選択中のパネルを削除 |
| パネルの詳細設定 | パネルを **マウス中ボタン（ホイール押し込み）でクリック** | そのパネルの機能別設定 popup を開く（[下記](#機能別設定-popup)） |

## 機能別設定 popup {#機能別設定-popup}

パネルを **中ボタンでクリック** すると、そのパネル内の文字やアイコンの大きさを個別に調整できる popup が開きます。

| 項目 | 内容 |
|---|---|
| 文字サイズ | パネル内の基本文字サイズ |
| 文字・路線サイズ | マップ内の文字・路線の太さ |
| 列車アイコン | マップの列車アイコンの大きさ |
| 駅アイコン | マップの駅アイコンの大きさ |
| 信号アイコン | マップの信号アイコンの大きさ |

- **値を変える**: 各項目の **数値にマウスカーソルを合わせてマウスホイール**（上で増える／下で減る）。**0 にすると「自動（推奨）」** になり、モニターとパネルの大きさから最適値が自動計算されます。
- **すべて自動に戻す**: **「✨ おすすめ (自動)」ボタンをクリック**。
- **閉じる**: **「閉じる」ボタンをクリック**。

## エディタ下部のボタン

| ボタン | 操作 | 動作 |
|---|---|---|
| 🗑 全消去 | クリック | 配置したパネルを全部消す |
| ✨ おすすめ | クリック | 推奨レイアウトを自動で配置する |
| ✓ 保存 | クリック | 現在のレイアウトを保存して閉じる |

## レイアウト保存

「✓ 保存」で、レイアウトは管理用コンピューター内に保存されます。  
[モニター連携カード](../tools/monitor-link-card.md) でリンクされた全モニターに、同じレイアウトが適用されます。
