# 路線記号タブ

![](bws:trainsystemutilities:wiki/screens/management-computer__symbol__ja_jp.png)

管理用コンピューターの路線記号タブ。路線記号 (例: `JA`, `JB`, `M01`) の作成・編集・削除を行う。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを **設置** して **右クリック** で画面を開く。
2. 左上のドロップダウンを **クリック** し、**「Ⓜ 路線記号」** を選ぶ。
3. 記号が並びきらないときは、一覧の上で **マウスホイール** を回してスクロールします。

## 路線記号とは

各駅・各列車に割り当てる識別記号。例: 山手線 = `JY`、中央線 = `JC`。  
[駅タブ](stations.md) で駅に割当 → [路線マップ](route-map.md) と [鉄道管理ブロック](../railway-management.md) に表示。

## 表示内容

| 列 | 内容 |
|---|---|
| 記号テキスト | 2-3 文字の路線略号 (例: `JA`) |
| 色 | 背景色 / 文字色 |
| 形状 | 円 / 角丸四角 / 六角形 など |
| 使用駅数 | この記号が割当られた駅の数 |

## 操作

| 操作 | やり方 | 動作 |
|---|---|---|
| 新規作成 | 右上の **「＋ 新規作成」ボタンをクリック** | [記号エディタ](symbol-editor.md) が開き、新しい記号を作れる |
| 編集 | 一覧の **記号タイルを（左）クリック** | その記号を [記号エディタ](symbol-editor.md) で編集 |
| 削除 | 一覧の **記号タイルを右クリック** | 削除確認が出る（[下記](#記号削除時の挙動)） |

> [!TIP]
> 記号タイルの上にマウスを乗せると「クリック: 編集 / 右クリック: 削除」という案内が表示されます。

## 記号削除時の挙動 {#記号削除時の挙動}

記号タイルを **右クリック** すると **削除確認** が表示されます。**「🗑 削除する」をクリック** で確定、そのまま閉じれば削除しません。

使用中の記号を削除すると、割当済みの全駅から記号がクリアされます。  
[駅タブ](stations.md) で再割当が必要です。
