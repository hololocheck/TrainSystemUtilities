# 路線マップ

![](bws:trainsystemutilities:wiki/screens/management-computer__map__ja_jp.png)

管理用コンピューターの路線マップタブ。全駅・全線路を 2D マップ表示する。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを **設置** して **右クリック** で画面を開く。
2. 左上のドロップダウンを **クリック** し、**「📁 自分の路線マップ」** を選ぶ（起動直後は既定でこのタブが開いています）。
3. マップは、事前に [メモリーカード](../tools/memory-card.md) で線路網をリンクしておくと表示されます。

## 表示要素

| 要素 | 表示 |
|---|---|
| 駅ノード | 円 + 駅名 |
| 線路エッジ | 駅間を結ぶ線 |
| 列車位置 | リアルタイム小アイコン |
| 信号機 | 状態 (赤/青) |

## 操作

マップは **移動（パン）/ 拡大縮小（ズーム）/ アイコンのクリック** で操作します。

| 操作 | やり方 | 動作 |
|---|---|---|
| パン（移動） | マップ上で **マウス左ボタンを押したままドラッグ** | 表示範囲を上下左右に動かす |
| ズーム（拡大縮小） | マップ上で **マウスホイールを回す** | 上に回すと拡大、下に回すと縮小 |
| 列車を選ぶ | **列車アイコンを左クリック** | その列車の **詳細ポップアップ** が開く（マップの上に重なって表示され、速度・行き先・時刻表などを確認できる） |
| 駅を選ぶ | **駅アイコンを左クリック** | **駅タブ** に切り替わり、その駅の詳細（路線記号の割当など）が開く |

> [!TIP]
> クリックは「ドラッグせずにその場で押して離す」操作です。少しでも動かすとパン（移動）になります。アイコンのすぐ近くを狙って、ちょんと押して離してください。

## 路線記号表示

駅ノードには [路線記号](line-symbols.md) が表示される (割当済みの場合)。  
[駅タブ](stations.md) で割当変更後にマップに即時反映。
