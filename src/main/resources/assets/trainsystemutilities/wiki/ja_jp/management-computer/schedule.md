# 電子式時刻表タブ

![](bws:trainsystemutilities:wiki/screens/management-computer__schedule__ja_jp.png)

管理用コンピューターの電子式時刻表タブ。列車の Create スケジュールを電子的に管理し、編集・物理アイテムへの書き出し・他列車との共有ができる。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを **設置** して **右クリック** で画面を開く。
2. 左上のドロップダウンを **クリック** し、**「🕒 電子式時刻表」** を選ぶ。
3. 列車が一覧に収まらないときは、リストの上で **マウスホイール** を回してスクロールします。

> [!NOTE]
> このタブの上部には **「⏹ 全列車停止」／「▶ 全列車再開」ボタン** もあります。クリックすると、この路線網の全列車をまとめて停止／再開できます（駅に着いた列車から順に停止します）。

## 時刻表の 3 つの状態

列車タイルには、その列車の時刻表の状態が表示される。

| 状態 | 意味 |
|---|---|
| 電子式時刻表 | この管理用コンピューターで設定・管理。編集 / 書き出し / 共有ができる |
| 通常の時刻表 | 物理スケジュールアイテムを運転士に直接渡したもの。コンピューターでは編集できない |
| なし | 時刻表が設定されていない |

## 詳細表示

![](bws:trainsystemutilities:wiki/screens/management-computer__schedule-detail__ja_jp.png)

**一覧の列車タイルをクリック** すると、その列車の詳細が開きます。

| 要素 | 操作 | 内容 |
|---|---|---|
| 運行再開 / 停止 | **クリック** で切替 | 停止中は「運行再開」、運行中は「停止」と表示されるトグルボタン |
| 編集 | **クリック** | 電子式時刻表を編集（条件を満たすと有効。[下記](#編集の条件)） |
| 🔗 共有 | **クリック** | 時刻表を他の列車へ共有（[共有](#share)） |
| ◀ 戻る | **クリック** | 詳細を閉じて一覧へ戻る |
| 明細 | 一覧の上で **マウスホイール** でスクロール | スケジュールエントリ一覧 |

### 編集の条件 {#編集の条件}

編集ボタンが有効になるには、次のすべてが必要:

- 列車が **停止中** であること
- **電子式時刻表** であること（通常の時刻表は編集不可）
- **運転士**（モブ または ブレイズバーナー）が搭乗していること

> [!TIP]
> 条件を満たしていないと、編集ボタンの文字が理由を示す表示に変わります。**「編集不可」** = 通常の時刻表、**「要運転士」** = 運転士が乗っていない、**「共有中」** = 他列車から共有された読み取り専用、の意味です。この状態ではクリックしても編集画面は開きません。

### 運転士の右クリック

電子式時刻表の列車は、運転士を右クリックすると **運行再開 / 停止のトグル** になり、**時刻表アイテムは取り出せない**（電子的に設定されているため）。通常の時刻表は従来どおり右クリックでアイテムを取り出せる。

## スケジュール編集 popup {#sched-editor}

![](bws:trainsystemutilities:wiki/screens/management-computer__schedule-editor__ja_jp.png)

詳細の **「✎ 編集」ボタンをクリック** すると、画面の左隣（入りきらない場合は右隣）に編集 popup が開きます。Create スケジュールの全命令・条件を Create 正式名称で扱えます。

### エントリ操作

- **エントリを追加**: popup 下部の **「動作を追加」ボタンをクリック** → 出てきた一覧（🚉 駅指定 / 📦 配送 / 📥 回収 / 📝 タイトル変更 / 🔧 速度制限）から **クリックで選ぶ**。
- **並べ替え**: 各エントリの **上向き矢印 / 下向き矢印をクリック** で 1 つ上／下へ移動。
- **削除**: エントリの **✕ をクリック**。
- **循環 / 片道の切替**: エディタ上部の **「↻ 循環運行」/「→ 片道運行」トグルをクリック**（循環にすると最後まで行ったら先頭へ戻ってループします）。
- **確定 / 取消**: 下部の **「✓ 適用」をクリックで保存**、**「× キャンセル」をクリックで破棄**。

### 各エントリの待機条件

各エントリには 1 つ以上の **待機条件** を割り当てられます。

- **条件を追加**: エントリの **条件追加ボタンをクリック** → 一覧（⏱ 待機 / ⌚ 時刻指定 / 👤 プレイヤー数 / 📦 アイテム / 💧 液体 / 🚃 連結 など）から **クリックで選ぶ**。
- **条件の数値を変える**: 待機秒数などの **数値にマウスカーソルを合わせて、マウスホイールを回す**（上で増える／下で減る）。
- **条件を削除**: 条件の **✕ をクリック**。

例:

| 条件 | 動作 |
|---|---|
| `⏱ 待機` | 指定時間待機 |
| `👤 プレイヤー数` | 乗客が乗るまで待機 |
| `📦 アイテム` / `💧 液体` | 貨物が積まれるまで待機 |
| `🚃 連結` | 連結待機（[連結 / 切り離し](../trains/coupling.md)） |

### エントリ追加（駅の選び方）

「動作を追加」で **🚉 駅指定 / 📦 配送 / 📥 回収** を選ぶと、続けて駅を選ぶ一覧（station-pick）が開きます。**行き先の駅をクリック** すると、そのエントリが追加されます。駅が多いときは一覧の上で **マウスホイール** でスクロールします。

## 時刻表に書き出し {#export}

電子式時刻表を、物理の「スケジュール」アイテムへ書き出せます。

- 搬入スロットに **空のスケジュールアイテムをドラッグして入れる** → **矢印型のバー** が進捗 → 搬出スロットに記入済みアイテムが出るので、**取り出す**。
- **「すべて書き出し」トグルをクリックして ON** にすると、搬入した数だけまとめて書き出します（OFF は 1 枚だけ）。

## 他の列車と共有 {#share}

![](bws:trainsystemutilities:wiki/screens/management-computer__schedule-share__ja_jp.png)

電子式時刻表を、**同一ネットワークの他の列車** と共有できます。

- 詳細の **「🔗 共有」ボタンをクリック** → 列車一覧が開く → 共有先の列車の行にある **トグルをクリックして ON**（緑）にすると、その列車が共有先になります。列車が多いときは一覧の上で **マウスホイール** でスクロールします。
- 共有先の列車は **読み取り専用** になり、「（列車名）と時刻表を共有しています」と表示される。
- 共有元を編集すると、共有先の時刻表へ **自動的に伝播** する。
