# 路線記号エディタ

```embed:symbol-editor
```

[路線記号タブ](line-symbols.md) から起動する記号エディタ popup。記号テキスト・色・形状を編集する。 色を選ぶ **カラーピッカー** もこのエディタの機能です ([下記](#color-picker))。

[[TOC]]

## 開き方

1. **管理用コンピューター** ブロックを右クリックして開く。
2. 左上のドロップダウンから **路線記号** を選ぶ ([路線記号タブ](line-symbols.md))。
3. 右上の **「＋新規作成」** ボタンで新規作成。 一覧の既存記号を **左クリック** すると編集、 **右クリック** で削除。

## 編集項目

各項目は値の欄を **クリック**、 または欄に **カーソルを合わせてマウスホイール** で変更します。 **色の欄をクリック** するとカラーピッカー ([下記](#color-picker)) が開きます。

| 項目 | 内容 | 操作 |
|---|---|---|
| 記号テキスト | 2-3 文字 (例: `JA`, `M01`) | 欄をクリックして入力 |
| 形状 | 円 / 角丸四角 / 六角形 / ダイヤ | クリックで切替 |
| 背景色 | 記号の下地色 | 色欄をクリック → カラーピッカー |
| 文字色 | テキストの色 | 色欄をクリック → カラーピッカー |
| 枠線色 | 縁取りの色 | 色欄をクリック → カラーピッカー |
| 枠線太さ | 0 / 1 / 2 / 3 px | クリックで切替 |
| フォント | 標準 / 太字 | クリックで切替 |

## プレビュー

エディタ右側にリアルタイムプレビュー。 変更が即時反映される。

## 既定の記号セット

新規作成時の初期テンプレート:
- 山手線風: `JY` (緑円)
- 中央線風: `JC` (橙円)
- 銀座線風: `G` (橙角丸)
- 丸ノ内線風: `M` (赤円)

## 保存と適用

**保存** ボタン → [路線記号タブ](line-symbols.md) の一覧に追加 → [駅タブ](stations.md) で駅に割当できます。

## カラーピッカー {#color-picker}

記号エディタの **色の欄 (背景色 / 文字色 / 枠線色) をクリック** すると開く、 色を選ぶための popup です。 記号の色だけでなく、 [色設定](color-settings.md)・[レイアウトエディタ](layout-editor.md)・[鉄道管理ブロックの色設定](../railway-management/color.md) でも同じピッカーが使われます (色プレビューをクリックすると開く)。

```embed:color-picker
```

### 入力モード

| モード | 内容 | 操作 |
|---|---|---|
| HSV カラーホイール | 色相環 + 明度スライダー | 円をクリック/ドラッグ |
| RGB スライダー | 赤 / 緑 / 青 0-255 | スライダーをドラッグ |
| HEX 入力 | `#RRGGBB` 形式 | 欄をクリックして入力 |
| プリセット | 12 色の標準色 | クリックで即適用 |

### 履歴・プレビュー・適用

- 直近使用した色 8 個を **履歴** に表示。 クリックで再適用。
- 選択中の色は **プレビュー** 領域に即時反映。
- **適用** ボタンを押すまで実際の色設定は変わりません (押さずに閉じれば元のまま)。
