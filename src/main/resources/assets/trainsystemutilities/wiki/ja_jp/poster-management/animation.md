# ポスター アニメーション設定

![](bws:trainsystemutilities:wiki/screens/poster-management-anim__ja_jp.png)

ポスター管理ブロックのスライド遷移を制御する popup。

[[TOC]]

## 開き方

1. [ポスター管理ブロック](../poster-management.md) を **右クリック** して GUI を開く。
2. **「♫ アニメーション」ボタンをクリック** すると、 このアニメーション設定 popup がダイアログの右側に出ます。
3. もう一度「♫ アニメーション」ボタンを押すと閉じます (Esc でも閉じられます)。

## 操作 (どこをクリック / ホイールするか)

- **アニメ種別**: 一覧から **使いたい種別 (「スライド左」「フェード」など) をクリック** すると選択されます (選択中はハイライト表示)。
- **間隔・速度の数値**: 値の表示に **カーソルを合わせてマウスホイール** で増減します (画面にも「↑↓ ホイール」とヒントが出ます)。 ＋ / − ボタンはありません。

## アニメーション種別

| タイプ | 動作 |
|---|---|
| **None** | 即時切替 (アニメなし) |
| **Slide Left/Right** | 左右スライドイン |
| **Slide Up/Down** | 上下スライドイン |
| **Fade** | フェード遷移 |
| **Zoom** | ズーム拡縮 |
| **Push Left/Right** | 押し出し |

## 設定項目

| 項目 (表示名) | 操作 | 用途 |
|---|---|---|
| アニメ種別 | 一覧を **クリック** | 遷移演出 (下記一覧から選択) |
| 間隔 (interval) | 値にカーソル → **ホイール** | スライド 1 枚あたりの表示時間 |
| 速度 (duration) | 値にカーソル → **ホイール** | 遷移アニメ自体の所要時間 |

## プレビュー

popup 内のプレビュー領域でリアルタイム確認可能。
