# クリエイターセンター

自分の投稿の詳細 analytics を表示するダッシュボード。

[[TOC]]

## 開き方 {#open}

クリエイターセンターは **自分の[プロフィール](profile.md)ページ** から開きます。

1. [列車プリセットツール](../train-preset-tool/browse.md)を **GUI モード** で **右クリック** して閲覧画面を開きます (モード切替は **Alt + ホイール**)。
2. モードを **`Place` (公開)** にし、上部の自分の名前/アイコン部分を **左クリック** して自分のプロフィールを開きます。
3. プロフィール内の **「クリエイターセンター」ボタン** を **左クリック** します。

> [!NOTE]
> クリエイターセンターは自分のアカウント専用です。他ユーザーのプロフィールからは開けません (ボタンが表示されません)。

## 表示内容

| 項目 | 概要 |
|---|---|
| 投稿数 | 公開中のプリセット総数 |
| 累計 DL | 全プリセットの累計ダウンロード |
| 累計いいね | 全プリセットの累計♥ |
| フォロワー | 自分をフォローしているユーザ数 |
| 月次ランキング | 自分の投稿の月次順位 |

## 認証管理

Microsoft アカウント認証の状態 / トークン再生成 / ログアウト。  
アカウント切替もここから。

## モデレータ機能 (限定)

OfficialAccounts のメンバーには追加機能:
- 通報されたプリセットのレビュー
- ユーザ BAN / unBAN
- アナウンス投稿
