# プリセット詳細

![](bws:trainsystemutilities:wiki/screens/preset-place-detail__ja_jp.png)

公開プリセットの個別ページ。3D プレビュー / 画像 / Markdown 説明 / ダウンロードを提供。

[[TOC]]

## 開き方 {#open}

1. [列車プリセットツール](../train-preset-tool/browse.md)を **GUI モード** で **右クリック** して閲覧画面を開きます (モード切替は **Alt + ホイール**)。
2. モードを **`Place` (公開)** に切り替えます。
3. 一覧のプリセットタイルを **左クリック** すると、このページが開きます。

## ボタンの操作 {#buttons}

各ボタンを **左クリック** で操作します。

| ボタン | 操作 | 動作 |
|---|---|---|
| ⬇ ダウンロード | 左クリック | プリセットを自分のローカルに保存 (下記 [ダウンロード](#download)) |
| ♡ / ♥ いいね | 左クリック | いいねの ON / OFF を切り替え |
| 3D | 左クリック | 3D プレビューを別窓 (モーダル) で開く (下記 [3D プレビュー操作](#p3d)) |
| 通報 | 左クリック | 通報ダイアログを開く (下記 [通報](#report)) |
| フォロー | 左クリック | 投稿者をフォロー / 解除 |
| 投稿者名 | 左クリック | その投稿者の[プロフィール](profile.md)へ移動 |
| × / Esc | クリック / キー | ページを閉じる |

## 主要要素

| 要素 | 機能 |
|---|---|
| 3D プレビュー | 列車構造を 3D 表示 (左ドラッグ回転 / 右ドラッグ pan / ホイール zoom) |
| 画像ギャラリー | アップロード時に添付された画像群 |
| Markdown 説明文 | クリエイターが書いた詳細解説 (4096 char) |
| ダウンロード ボタン | ローカルにダウンロード保存 |
| いいね (♥) | プリセットへの評価 |
| 通報 ボタン | 不適切時に理由付きで報告 |
| アップローダ情報 | クリエイター名 + フォロー ボタン |

## 3D プレビュー操作 {#p3d}

**3D** ボタンを押すと、列車構造を回して見られる別窓 (モーダル) が開きます。窓の中で次の操作ができます。

| 操作 | 動作 |
|---|---|
| 左ドラッグ | XP/YP 軸回転 |
| 右ドラッグ | 平行移動 (pan) |
| ホイール | zoom (0.2× ~ 8×) |
| ESC / 背景クリック | 閉じる |

## ダウンロード {#download}

**⬇ ダウンロード** ボタンを **左クリック** すると、自分のローカルストレージに保存されます。  
保存後は [列車プリセット閲覧 > Mine モード](../train-preset-tool/browse.md) から配置できます。

## 通報 {#report}

![](bws:trainsystemutilities:wiki/screens/preset-place-detail-report__ja_jp.png)

通報ダイアログで理由 (500 文字) を入力 → 送信。  
TSU モデレータがレビューして対応。
