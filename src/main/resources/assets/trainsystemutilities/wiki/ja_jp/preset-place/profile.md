# プロフィール

![](bws:trainsystemutilities:wiki/screens/preset-place-profile__ja_jp.png)

ユーザー個別ページ。自分のページと他ユーザーページの 2 モード。

[[TOC]]

## 開き方 {#open}

- **自分のプロフィール** … [列車プリセットツール](../train-preset-tool/browse.md)を **GUI モード** で **右クリック** → モードを **`Place` (公開)** にし、上部の自分の名前/アイコン部分を **左クリック**。
- **他ユーザーのプロフィール** … [プリセット詳細](detail.md)ページで、**投稿者名を左クリック** すると開きます。

初めて開くときは Microsoft アカウントの認証が求められることがあります。画面の案内に従ってください。

## モード {#modes}

自分のページか他人のページかで、できることが変わります。

| モード | 内容 |
|---|---|
| **自分** | プロフィール編集 / アイコン変更 / 統計確認 / クリエイターセンターへ |
| **他人** | 公開プリセット閲覧 / フォロー |

## ボタンの操作 {#buttons}

| ボタン | 表示条件 | 操作・動作 |
|---|---|---|
| フォロー | 他人のページ | **左クリック** でフォロー / 解除 (フォロー中はグレー、未フォローはオレンジ) |
| クリエイターセンター | 自分のページ | **左クリック** で [クリエイターセンター](creator-center.md) を開く |
| アイコン編集 | 自分 かつ 公式アカウントのみ | **左クリック** でアイコンエディタを開く |
| プリセットタイル | 両方 | **左クリック** でそのプリセットの[詳細](detail.md)へ |
| × / 戻る | 両方 | ページを閉じる / 戻る |

## プロフィールアイコン {#icon}

SVG 形式のカスタムアイコンに対応。  
編集はプロフィールアイコン エディタ (専用 popup。自分のページの **アイコン編集** ボタンから開く) で行います:

- SVG ファイル読込
- scale / offset の調整
- 背景色 / 額縁色 設定
- **公式アカウント (OfficialAccounts) のみ編集可** (一般ユーザーには編集ボタンが表示されません)

## 統計表示

自分のプロフィールでは以下が見える:
- 公開プリセット数
- 総ダウンロード数
- 総いいね数
- フォロワー数
