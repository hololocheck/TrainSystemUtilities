# アナウンス設定 (SAS 連携)

![](bws:trainsystemutilities:wiki/screens/railway-management-announcement__ja_jp.png)

[StationSoundSystem (SAS)](https://github.com/hololocheck/SpatialAudioSystem) 連携時に表示される popup。  
列車の到着 / 出発 / 通過等のイベントに紐づく発車メロディ・案内放送・ジングルを管理。

[[TOC]]

> [!IMPORTANT]
> このページは **SpatialAudioSystem (SAS) MOD が同時導入されている場合のみ** 機能する。  
> SAS 未導入時は「機能 ▼」一覧に「アナウンス」が出ず、 popup 自体が開けない。

## 開き方

1. [鉄道管理ブロック](../railway-management.md) を **右クリック** して GUI を開く。
2. モニター行の **「機能 ▼」ボタンをクリック**。
3. 出てきた一覧から **「アナウンス」をクリック** すると、 このアナウンス設定 popup が開きます (SAS 導入時のみ「アナウンス」が表示されます)。

## トップ設定

各トグルは **クリック** で ON / OFF が切り替わります。

| 項目 (表示名) | 操作 | 用途 |
|---|---|---|
| 「検知有効」トグル | クリック | アナウンス全体の ON / OFF (マスタースイッチ) |
| 「範囲枠表示」トグル (client-side) | クリック | 検知範囲を境界線で可視化 (自分の画面だけ。 位置合わせ用) |
| 「減衰モード」トグル | クリック | ON = 距離減衰、 OFF = 駅近辺一律 |

## エントリ管理

エントリ = (条件) → (再生する音声) のペア。  
1 駅で複数エントリを順番に持ち、条件がマッチしたエントリを再生します。 音声は各エントリのスロットに **SAS の記憶媒体アイテムを入れる** ことで割り当てます。

### エントリの追加・削除・テスト

- **「+ エントリを追加」ボタンをクリック**: 新しいエントリを追加。
- **「テスト再生」ボタンをクリック**: 動作確認用に再生。
- エントリが多いときは、 一覧の上で **マウスホイール** でスクロールします。

### 条件を選ぶ (dropdown)

![](bws:trainsystemutilities:wiki/screens/railway-management-announcement__ja_jp.png)

各エントリの **条件表示 (▾ 付き) をクリック** すると一覧が開き、 次の 3 つから **クリック** で選べます。 条件表示を **右クリック** すると「なし」に戻せます。

| 条件 (表示名) | トリガ |
|---|---|
| なし | 何もしない (無効なエントリ) |
| 通過時 | 検知範囲を列車が **通過** した瞬間 |
| 停止時 | 検知範囲内に列車が **停止** した瞬間 |

### タイミングと再生回数

各エントリの以下の値は、 **値にカーソルを合わせてマウスホイール** で増減します。

| 項目 (表示) | 操作 | 内容 |
|---|---|---|
| 遅延 (`↕ +Ns`) | 値にカーソル → **ホイール** | 条件成立から再生までの遅延秒数 (マイナスも可) |
| 再生回数 (`↕xN`) | 値にカーソル → **ホイール** | そのエントリを繰り返す回数 |

### 音声の割り当て (記憶媒体スロット)

popup 下部に **「検知カード」** と **「範囲指定ボード」** の 2 つのアイテムスロットがあります。 ここに SAS の音声を持つアイテムを **通常のインベントリ操作 (クリックで持ち上げて置く)** で入れると、 そのエントリの再生音声になります。

## アナウンス共有 (Share)

popup の **「共有」ボタンをクリック** すると「他の駅と共有」一覧が開き、 他の駅 (= [管理用コンピューター](../management-computer/overview.md) に登録された駅) にこの駅のアナウンス設定を共有できます。

一覧の各駅の右にある 2 つのトグルを **クリック** して、 駅ごとに個別 ON / OFF します。

| トグル (表示名) | 共有される内容 |
|---|---|
| 「検知」 | 検知カード (検知範囲) の設定を共有 |
| 「範囲指定」 | 範囲指定ボード (動作範囲) の設定を共有 |

- 駅が多いときは一覧の上で **マウスホイール** でスクロールします。
- 逆に他の駅から共有を **受けている** ときは、 popup 上部に「〇〇 から共有」と表示されます。
