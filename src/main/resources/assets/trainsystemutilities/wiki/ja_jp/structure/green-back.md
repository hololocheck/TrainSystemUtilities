# グリーンバック (Green Back)

```embed:item id=trainsystemutilities:green_back size=48 label=true
```

クロマキー合成用の純緑 (#00FF00) ブロックです。常に明るさ 15 で発光するため、影が落ちず均一な緑になります。

## 設置のしかた {#place}

- 通常のブロックと同じく、手に持って **右クリック** で設置します。開く GUI や特別な操作はありません。
- 撤去は他のブロックと同じく破壊するだけです。背景として広い面に **敷き詰めて** 使います。

## 用途

- 車両や建築物の紹介動画・スクリーンショットの背景に敷き詰めて、後から映像編集ソフトで背景を抜きます。
- [プリセット投稿](../preset-place/upload.md)用のスクリーンショット撮影背景にも便利です。

> [!TIP]
> 被写体の真下にも敷くと、下面への緑かぶりは増えますが切り抜きは楽になります。用途に応じて使い分けてください。
