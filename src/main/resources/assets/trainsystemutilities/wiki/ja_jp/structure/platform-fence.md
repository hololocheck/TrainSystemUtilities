# ホームフェンス (1m / 3m / 5m)

```embed:items size=32 cols=3 label=true ids=trainsystemutilities:platform_fence_1m,trainsystemutilities:platform_fence_3m,trainsystemutilities:platform_fence_5m
```

駅ホームの縁に設置する転落防止フェンスです。長さ 1m / 3m / 5m の 3 種類があり、帯の色を路線カラーに合わせて動的に変更できます。

[[TOC]]

## 設置のしかた {#place}

1. インベントリでホームフェンス (1m / 3m / 5m のいずれか) を手に持ちます。
2. ホームの縁に向けて **右クリック** すると設置されます。
3. フェンスは **自分から見て左右に伸びる向き** で置かれ、帯の面がこちら側を向きます。設置したい向きに体を向けてから右クリックしてください。

> [!NOTE]
> 3m / 5m は複数マスにまたがる 1 つのブロックです。**伸びる先が塞がっていると設置できません** (アイテムは消費されません)。前方に空きマスを確保してから置いてください。1m は 1 マスなので狭い場所でも置けます。

> [!TIP]
> 直線が長い区間は 5m、細かい調整が必要な箇所は 1m を組み合わせると綺麗に収まります。

## 帯の色について {#band-color}

帯色は[路線記号](../management-computer/line-symbols.md)や[帯色設定](../railway-management/color.md)と同じ色体系で表示され、路線のブランドカラーに揃えられます。初期色は緑 (山手線風) です。

帯色は **フェンス単体を右クリックしても変わりません**。色は [ホームドア](platform-screen-door.md) と同じ **メモリーカードのグループ + 鉄道管理ブロックの「帯色変更」アクション** で一括設定します。手順は次の通りです。

1. **メモリーカード** を手に持ち、色を揃えたいフェンス (とホームドア) を **右クリック** してグループ登録します (下記 [グループ登録](#group))。
2. 駅の[鉄道管理ブロック](../railway-management.md)を開き、そのメモリーカードを **ホームドア用スロット** に入れます。
3. ホームドア設定で **帯色** を選び、アクションが「帯色変更」の条件を追加します。列車が到着/発車したとき、グループ内のフェンス・ドアの帯色がまとめて変わります。

詳しい設定画面は [ホームドア](platform-screen-door.md) のページを参照してください (フェンスとホームドアはまったく同じ仕組みで色・グループを共有します)。

## メモリーカードでのグループ登録 {#group}

複数のフェンス/ドアを 1 つの「グループ」としてまとめ、帯色などを一括で扱うために使います。

1. **メモリーカード** を手に持ちます。
2. グループに入れたいフェンス (またはホームドア) のどれか 1 つを **右クリック** します。
3. **隣り合って連結しているフェンス/ドアが自動でまとめて登録** されます (1 個ずつ登録する必要はありません)。チャットに「ホームドア群に ○個 追加 (計 ○個)」と表示されます。
4. カードを持っている間、登録済みメンバーが **緑の枠** でワールド上に表示されるので、範囲を目で確認できます。
5. 余分なメンバーを外したいときは、その部分を **Shift + 右クリック** します (「○個 削除」と表示)。
6. カードの現在の登録数は、アイテムの **説明 (tooltip)** で確認できます。

> [!NOTE]
> ここで作るグループは、ドアの自動開閉や帯色変更のために鉄道管理ブロックへ渡す「対象リスト」です。使い方の続きは [ホームドア > 駅と連動させる](platform-screen-door.md#link) を参照してください。
