# 券売機 (Ticket Vending Machine)

```embed:item id=trainsystemutilities:ticket_vending_machine size=48 label=true
```

駅のきっぷ券売機です。高さ 2 ブロックの筐体で、右クリックすると実駅の券売機のような UI が開き、行き先を選んで切符を発券できます。

[[TOC]]

## 設置と駅の紐付け {#place}

1. 券売機アイテムを手に持ちます。
2. 設置したい場所に向けて **右クリック** で設置します (筐体は高さ 2 ブロックぶんの空間が必要です。上に 1 ブロックの空きを確保してください)。正面が自分の方を向くように置かれます。
3. **[駅範囲指定ツール](../tools/station-range-tool.md) で作った駅グループの範囲内に設置すると、自動でその駅に紐付きます** (= その駅が発駅になります)。
4. 既に範囲がある場所に後から設置しても接続されます。 範囲を後から作った場合も、 券売機を次に開いたときに紐付けし直します。

> [!WARNING]
> **駅範囲外に置いた券売機は使えません。** 右クリックしても UI は開かず、「駅の範囲内に設置してください」と赤字で表示されます。必ず駅グループの範囲内に設置してください。駅グループの作り方は [駅範囲指定ツール](../tools/station-range-tool.md) を参照。

## 券売機 UI の開き方と発券 {#open}

設置した券売機を **右クリック** すると券売機 UI が開きます。

- 行き先 (= 販売可に設定された駅) が角丸のボタンで並びます。数が多いときは **マウスホイール** でスクロールできます。
- 行きたい駅の **ボタンを左クリック** すると **切符** が発券され、インベントリに入ります (v1 は **無料**)。
- 並ぶ行き先は、この券売機と **同じ線路網につながっている駅** のうち、販売可に設定されたものだけです (自駅は除外)。
- ヘッダーは BelugaExperience 標準です (**× で閉じる** / **ヒントトグル** / **📖 で wiki**)。閉じるときは × ボタンか Esc キー。

## 切符

```embed:item id=trainsystemutilities:ticket size=32 label=true
```

発券された切符には **発駅・着駅** が記録され、 アイテムの説明 (tooltip) に「発: ○○ / 着: △△ (まで有効)」と表示されます。 v1 では情報アイテムで、 改札による検証は将来対応予定です。

## 販売する駅の設定

券売機に並ぶ行き先は、 **[管理用コンピューターの券売機タブ](../management-computer/tickets.md)** で駅ごとに販売可/不可を切り替えて決めます。 設定はネットワーク全体で共有され、 すべての券売機に反映されます。
