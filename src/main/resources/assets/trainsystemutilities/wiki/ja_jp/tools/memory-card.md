# メモリーカード

```embed:item id=trainsystemutilities:memory_card size=48 label=true
```

**管理用コンピューター** に線路網や駅ブロックをリンクしたり、 **ホームドア / ホームフェンス** をグループ登録するための汎用リンクカード。 位置を「登録」してから、 別のブロックに「適用」する 2 ステップで使います。

[[TOC]]

## 基本の使い方

1. **登録**: リンク元 (線路 / 鉄道管理ブロック / ホームドア等) を **右クリック** して位置をカードに記録。
2. **適用**: リンク先 (多くは管理用コンピューター) を **右クリック** して記録内容を反映。
3. カードの現在の登録内容は、 アイテムの **説明 (tooltip)** で確認できます。
4. **Shift + 右クリック** (何もない場所や対象外ブロック) で **カードを初期化** します。

## メモリーカードで操作できる対象

| 対象 | 右クリック | Shift + 右クリック |
|---|---|---|
| Create の **線路** | 線路ネットワークを登録 (駅数 / 信号数 / 列車数を記録) | (初期化) |
| **鉄道管理ブロック** | その駅ブロックを登録 (駅名付き) | (初期化) |
| **管理用コンピューター** | 登録済みの線路網 / 鉄道管理ブロックを **リンク** | (初期化) |
| **ホームフェンス / ホームドア** | 連結しているフェンス / ドアを **まとめてグループ登録** | そのメンバーを **グループから削除** |

> [!NOTE]
> **モニターへのリンクは専用の [モニター連携カード](monitor-link-card.md)** を使います (メモリーカードとは別アイテム)。 列車検知は [列車検知カード](train-detection-card.md)。

## よくある手順

**管理用コンピューターに路線網をリンクする**

1. 路線上の任意の **線路を右クリック** → 「線路ネットワークを登録」と表示。
2. **管理用コンピューターを右クリック** → 「線路ネットワークをリンクしました」。
3. これで管理用コンピューターの路線マップ / 列車一覧に路線網が表示されます。

**駅の鉄道管理ブロックをコンピューターに紐付ける**

1. 駅ホームの **鉄道管理ブロックを右クリック** で登録。
2. **管理用コンピューターを右クリック** でリンク。

**ホームドア / ホームフェンスをグループ化する**

1. ドア / フェンスの 1 つを **右クリック** すると、 隣接して連結しているブロックが自動でまとめて登録されます。
2. 余分なメンバーは **Shift + 右クリック** で除外。
3. グループの駅連動は [ホームドア](../structure/platform-screen-door.md) を参照。

## モード非依存

メモリーカードによるリンク / 読み取りはアクセスモード (Private/Public) に関係なく動作します。
