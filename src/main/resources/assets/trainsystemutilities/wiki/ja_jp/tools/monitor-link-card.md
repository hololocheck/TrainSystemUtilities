# モニター連携カード

```embed:item id=trainsystemutilities:monitor_link_card size=48 label=true
```

モニターブロックを別のブロック (鉄道管理 / ポスター管理 / 管理用コンピューター) にリンクする、**手に持って使う専用カード**。

[[TOC]]

## 持ち方 / 使い方

このカードには専用の GUI はありません。**手に持って、モニターを右クリックするだけ**で使います。

1. モニター連携カードを**ホットバーに入れて手に持ちます**。
2. リンクしたい**モニターブロックを右クリック**（初期設定では**マウス右ボタン**）します。
   - このとき、隣り合って連結しているモニターは**自動でまとめて 1 グループとして登録**されます（1 面ずつクリックする必要はありません）。
   - すでに登録済みのグループをもう一度右クリックすると、そのグループの**登録が解除**されます。
3. 登録すると、画面下部（ホットバー上）に「登録しました（○グループ）」と表示されます。
4. 登録した内容は、カードにマウスを乗せたときの**説明（ツールチップ）**で「登録済み: ○個」と確認できます。
5. **Shift + 右クリック**（しゃがみながら右クリック）すると、カードの**登録内容がすべて消去**されます。

> [!NOTE]
> このカードは「登録したモニター」を覚えるカードです。実際にモニターへ表示内容を出すには、  
> 登録済みのこのカードを**鉄道管理ブロック等のモニタースロットに入れます**（下記「使い方の流れ」を参照）。

## 操作まとめ

| 操作 | 何が起きるか |
|---|---|
| モニターを**右クリック** | そのモニター（連結グループ）を登録 / 登録済みなら解除 |
| **Shift + 右クリック** | カードの登録内容をすべて消去 |
| カードに**マウスを乗せる** | 現在の登録数をツールチップで確認 |

## 使い方の流れ

1. 上記の手順でモニターをカードに**登録**します。
2. 登録済みのカードを、表示元にしたいブロック（[鉄道管理ブロック](../railway-management.md) / [ポスター管理ブロック](../poster-management.md) / [管理用コンピューター](../management-computer/monitor.md)）の**モニタースロットに入れます**。
3. これで、そのブロックの表示内容が、登録したモニターへ即座に同期表示されます。

## リンク可能な組み合わせ

| コピー元 | モニター表示内容 |
|---|---|
| [鉄道管理ブロック](../railway-management.md) | 駅の到着 / 次列車 + 路線記号 |
| [ポスター管理ブロック](../poster-management.md) | スライドショー画像 |
| [管理用コンピューター モニター](../management-computer/monitor.md) | カスタムレイアウト |

## モニターブロックの設置

モニターは 1 ブロックでも、複数ブロック組合せた **多面モニター** (= 縦×横の grid) でも可。  
連携カードでリンクすると grid 全体に内容を引き伸ばして表示する。
