# 列車検知カード

```embed:item id=trainsystemutilities:train_detection_card size=48 label=true
```

線路上の**特定の 1 地点**を「列車が通過したら反応する場所」として記録する、**手に持って使う補助カード**。  
記録した地点は、[鉄道管理ブロック](../railway-management.md)の [SAS アナウンス](../railway-management/announcement.md)で「この地点を列車が通ったらアナウンスを鳴らす」トリガとして使います。

[[TOC]]

## 持ち方 / 使い方

このカードには専用の設定 GUI はありません。**手に持って線路を右クリックし、地点を記録するだけ**です。細かい条件（対象列車・アナウンス内容など）は、記録後に**鉄道管理ブロック側の GUI**で設定します。

1. 列車検知カードを**ホットバーに入れて手に持ちます**。
2. 記録したい**線路ブロック（Create のトラック）を右クリック**（初期設定では**マウス右ボタン**）します。
   - 線路以外を右クリックすると「線路ではありません」という趣旨のメッセージが出て、記録されません。
3. 記録に成功すると、画面下部に「地点を記録しました（座標）」と表示されます。
4. 記録した座標は、カードに**マウスを乗せたとき**の説明（ツールチップ）で確認できます。
5. **Shift + 右クリック**（しゃがみながら右クリック）すると、記録した地点が**消去**されます。

> [!NOTE]
> このカードが覚えるのは**線路上の 1 地点だけ**です。「検知範囲（半径）」や「対象列車の絞り込み」「発火タイミングの種類」といった細かい条件は**カード側にはありません**。  
> それらは、記録済みのカードを鉄道管理ブロックに入れたあと、[SAS アナウンス設定](../railway-management/announcement.md)の GUI 側で調整します。

## 操作まとめ

| 操作 | 何が起きるか |
|---|---|
| 線路を**右クリック** | その線路の地点を検知地点として記録 |
| **Shift + 右クリック** | 記録した地点を消去 |
| カードに**マウスを乗せる** | 記録済みの座標をツールチップで確認 |

## 鉄道管理ブロックでの使い方

記録済みのカードは、次の流れで実際のトリガにします。

1. 上の手順で、アナウンスを鳴らしたい線路の地点をカードに記録します。
2. [鉄道管理ブロック](../railway-management.md)を右クリックして開き、[SAS アナウンス設定](../railway-management/announcement.md)を開きます。
3. アナウンス設定内の**検知カード用スロットに、この記録済みカードを入れます**。
4. これで、記録した線路の地点を列車が通過したときに、設定したアナウンスが再生されます。

## 連携
