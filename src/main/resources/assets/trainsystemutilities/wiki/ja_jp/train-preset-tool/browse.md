# プリセット閲覧 / 配置

![](bws:trainsystemutilities:wiki/screens/train-preset-browse-mine__ja_jp.png)

保存済みプリセットの閲覧 / 設置 / アップロード / 削除 を行う、**列車プリセットツール**のメイン GUI です。

[[TOC]]

## 開き方 / 持ち方

この画面は **GUI モード**の列車プリセットツールで開きます（ツールを持ち替えた直後は GUI モードです）。

1. 列車プリセットツールを**手に持ちます**。
2. そのまま**右クリック**（初期設定では**マウス右ボタン**）します。この画面が開きます。
3. 閉じるときは **Esc キー**、または画面右上の**「×」を左クリック**します。

> [!NOTE]
> もし右クリックで別の画面（保存画面）や配置動作になる場合は、ツールが GUI 以外のモードになっています。  
> **Alt + マウスホイール**で **GUI モード**に戻してから右クリックしてください（現在のモードはホットバー上に表示されます）。

## モード切替 (dropdown)

画面上部の**モード表示（ドロップダウン）を左クリック**すると一覧が開き、**Mine / Place を左クリック**で切り替えます。

| モード | 表示内容 |
|---|---|
| **Mine** (自分の) | ローカル保存済みプリセット (= 自分が save したもの) |
| **Place** (公開された) | [Preset Place](../preset-place/overview.md) からダウンロード可能なオンラインプリセット |

## 主要操作

| 操作 | 動作 |
|---|---|
| タイルを**左クリック** | プリセット選択 → 右パネルに 3D プレビュー |
| タイルを**右クリック** | プリセット削除確認 (Mine) |
| タイルを**右パネルへドラッグ&ドロップ** | 設置の準備。プレビューと必要素材が出て「▶ 進む / ✖ キャンセル」ボタンが表示される |
| **「▶ 進む」**を左クリック | 設置を確定。この画面が閉じ、ツールが**配置（Place）モード**になる |
| **更新ボタン**を左クリック | 一覧を再読み込み |
| アップロードアイコンを左クリック | Preset Place へアップロード ([upload](../preset-place/upload.md)) |

## 3D プレビュー

右パネルに選択中プリセットを 3D で表示します。プレビューの上でマウス操作します:
- **左ドラッグ**: 回転
- **右ドラッグ**: 平行移動 (pan)
- **マウスホイール**: 拡大縮小 (zoom)

## 検索 (Place モード)

公開プリセットを名前で検索。上部の**検索欄を左クリック**して名前を入力し、下記ソートと組合せて絞り込みます:
- 作成日 / いいね順 / ダウンロード順

## 素材表示

選択中プリセットに必要なブロック / アイテムを一覧表示します。  
不足分は**赤**、充足分は**緑**で表示。詳細は [素材補充](refill.md)。

## 素材の補充元 (Material source)

必要素材をどこから引き出すかを、右パネルの**補充元ピル（pill）ボタンの上でマウスホイール**を回して切り替えます:

| 補充元 | 動作 |
|---|---|
| **Chest** | リンクしたチェストから自動で引き出す |
| **ME** | AE2 のネットワークから引き出す |

> [!TIP]
> チェストのリンクは、ツールを持った状態で狙ったチェストに **Shift + 中クリック**します（GUI を閉じた通常の持ち手時に操作）。

## グルータンク

設置時に必要な Create のスーパーグルー（粘着剤）量がここに表示されます。

- **`充填` ボタンを左クリック** → [素材補充](refill.md)画面が開き、スライムボール等でタンクを補充できます。
- **`ダンプ` ボタンを左クリック** → タンクの中身を全廃棄します。

## 設置（配置）の流れ

「▶ 進む」で確定すると GUI が閉じ、ツールが**配置（Place）モード**になります。続いてワールドで設置します:

1. 列車を置きたい**線路の位置を右クリック**（初期設定では**マウス右ボタン**）して**起点**を決めます。
2. **Alt + ホイール**で向きを 90° ずつ回転できます（必要に応じて）。
3. もう一度**右クリック**で設置を実行します。素材と粘着剤がここで消費されます。
4. 設置に失敗して再開できる場合は**中クリック**で再試行、**Shift + 中クリック**で配置モードを**キャンセル**して GUI モードに戻れます。
