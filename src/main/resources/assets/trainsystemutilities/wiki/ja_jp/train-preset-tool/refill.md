# 素材補充

![](bws:trainsystemutilities:wiki/screens/train-preset-refill__ja_jp.png)

列車プリセットツールの**粘着剤（Create スーパーグルー）タンクを補充する GUI** です。プリセットを設置するとき、列車ブロックを接着するために粘着剤を消費します。その補給をここで行います。

[[TOC]]

## 開き方

この画面は、[プリセット閲覧 / 配置画面](browse.md)から開きます。

1. 列車プリセットツールを持ち、**右クリック**で[プリセット閲覧 / 配置画面](browse.md)を開きます（GUI モード時）。
2. 右パネルのグルータンク欄にある**`充填` ボタンを左クリック**します。
3. この補充画面が開きます。閉じるときは **Esc キー**、または右上の**「×」を左クリック**します。

## できること

| 機能 | 概要 |
|---|---|
| 粘着剤タンク補充 | スライムボール / スライムブロックを入れてタンクを満たす |
| タンク残量表示 | 現在量 / 最大量（例: `0 / 10000`）とゲージで表示 |
| ダンプ | タンクの中身を全廃棄 |

## 粘着剤の入れ方

画面中央の**入力スロット**に、**スライムボール**または**スライムブロック**を入れます（それ以外のアイテムは入りません）。

1. インベントリのスライムボール / スライムブロックを、入力スロットへ**左クリックで移動**します（**Shift + 左クリック**でまとめて移動もできます）。
2. 入れた瞬間、タンクに自動で加算されます（タンク容量に対して、スライムボール 1 個 = 10、スライムブロック 1 個 = 90 の量。最大容量は 10000）。
3. タンクが満タンに近く、1 個分の余裕もないときは加算されず、スロットにそのまま残ります。
4. 画面を閉じると、入力スロットに残ったアイテムは自動でインベントリに戻ります。

> [!NOTE]
> この画面で入れるのは**粘着剤（スライム）だけ**です。列車を構成するブロックそのもの（レール・板ガラス等）は、  
> ここではなく[プリセット閲覧 / 配置画面](browse.md)で指定した**補充元（Chest / ME）**から設置時に引き出されます。

## タンク残量とダンプ

- 画面上部に**現在量 / 最大量**とゲージが表示されます。
- **`ダンプ` ボタンを左クリック**すると、タンクの中身をすべて廃棄します。

## プリセット設置との連携

[プリセット閲覧 / 配置画面](browse.md)で 3D プレビューを右パネルにドロップ → 「▶ 進む」で配置を確定 → ワールドで設置したとき、ここで貯めた粘着剤が消費されます。  
粘着剤が足りないと設置時に赤字で警告が出て、設置できません。
